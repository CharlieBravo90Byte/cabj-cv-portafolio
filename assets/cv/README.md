# Carpeta para CV en PDF

## Instrucciones:

Coloca aquí tu CV en formato PDF con el nombre: **Carlos_Bravo_CV.pdf**

Este archivo será descargado cuando los visitantes hagan clic en el botón "Descargar CV" de tu página web.

### Recomendaciones para el PDF:
- Mantén el PDF actualizado con tu información más reciente
- Tamaño recomendado: máximo 2 MB
- Formato: A4 o Letter
- Asegúrate de que sea legible y profesional
