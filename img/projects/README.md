# Carpeta de Imágenes de Proyectos

## Instrucciones:

Coloca aquí las imágenes de tus proyectos con los siguientes nombres:

- **project1.jpg** - Sistema de Gestión Empresarial
- **project2.jpg** - API REST de Automatización  
- **project3.jpg** - Dashboard de Analítica
- **project4.jpg** - ETL con Talend

### Recomendaciones:
- **Formato:** JPG o PNG
- **Dimensiones:** Mínimo 800x600px (recomendado 1200x800px)
- **Tamaño:** Máximo 500KB por imagen
- **Contenido:** Screenshots del proyecto, diseño UI/UX, o mockups profesionales

### Alternativas si no tienes capturas:
- Usa imágenes placeholder de servicios como:
  - https://placeholder.com
  - https://picsum.photos
  - https://unsplash.com (imágenes tech relacionadas)

### Ejemplo de nombres temporales:
Mientras no tengas las imágenes reales, puedes usar estas URLs temporales en el HTML:
- https://via.placeholder.com/800x600/3b82f6/ffffff?text=Proyecto+1
